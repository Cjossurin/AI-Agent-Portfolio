# Content Type Guidelines & Specifications
## Master Reference for Prompt Templates

This document defines the optimal **length**, **formatting**, and **structure** for each content type across all platforms, broken down by goal (engagement, growth, sales).

---

## 📘 FACEBOOK

### Post (Social Media Post - NOT Blog Article)
| Goal | Length | Format | Key Elements |
|------|--------|--------|--------------|
| **Engagement** | 100-250 chars | Conversational, question-based | Hook question, emoji, engagement CTA |
| **Growth** | 150-400 chars | Value-driven, shareable | Value proposition, follow CTA, hashtags (3-5) |
| **Sales** | 150-300 chars | Problem→Solution→CTA | Pain point, benefit, urgency, link CTA |

**Format Rules:**
- Use line breaks for readability
- Bold text with **asterisks** or Unicode bold
- Emojis for visual breaks (2-4 max)
- NO markdown headers (#)
- NO horizontal rules (---)
- Short paragraphs (1-3 sentences)

### Caption (For photo/video posts)
| Goal | Length | Format | Key Elements |
|------|--------|--------|--------------|
| **Engagement** | 40-150 chars | Quick, punchy | Hook + engagement question |
| **Growth** | 100-300 chars | Value + CTA | What they'll get + follow CTA |
| **Sales** | 80-200 chars | Direct | Benefit + urgency + shop CTA |

### Ad
| Goal | Length | Format | Key Elements |
|------|--------|--------|--------------|
| **Engagement** | 90-125 chars primary | Native feel | Hook + soft CTA |
| **Growth** | 100-150 chars | Value proposition | Why follow + Page Like CTA |
| **Sales** | 125-150 chars | Direct response | Problem→Solution→Offer→CTA |

### Carousel
| Goal | Length | Format | Key Elements |
|------|--------|--------|--------------|
| **Engagement** | 3-5 slides, 20-50 chars each | Tips/list format | Value per slide, swipe CTA |
| **Growth** | 3-5 slides | Educational | Shareable insights, follow CTA |
| **Sales** | 3-5 slides | Problem→Solution journey | Benefit per slide, final CTA |

## 🎯 GOAL-SPECIFIC GUIDANCE

### Views/Engagement Goal
- **Focus:** Community, conversation, reactions
- **Tone:** Conversational, relatable, curious
- **CTA Type:** Questions, polls, "tag a friend", "share your..."
- **Avoid:** Sales language, links, promotional tone

### Follower/Growth Goal
- **Focus:** Value demonstration, shareability
- **Tone:** Authoritative yet generous
- **CTA Type:** "Follow for more", "Save this", "Share with..."
- **Avoid:** Hard sells, too much promotional content

### Conversions/Sales Goal
- **Focus:** Problem→Solution, benefits, urgency
- **Tone:** Direct, confident, benefit-focused
- **CTA Type:** "Shop now", "Get yours", "Limited time", links
- **Avoid:** Being too pushy, vague benefits

## ✅ UNIVERSAL FORMATTING RULES

**DO:**
- Use **bold** for emphasis (asterisks work on most platforms)
- Use line breaks for readability
- Use emojis strategically (not excessively)
- Use bullet points (•, -, ✅) for lists
- Front-load the hook/value

**DON'T:**
- Use # headers for social posts (blog only)
- Use --- dividers (blog only)
- Use [VISUAL: ...] tags in final output
- Use <XML> tags in final output
- Include scratchpad/planning notes
- Use markdown that won't render on platform
